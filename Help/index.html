Sobre InVendi

InVendi BMS es una aplicación para la gestión eficiente de edificios que permite la visualización y el control de sistemas. Mediante una interfaz sencilla permite crear interfaces gráficas modernas y personalizables adaptables a pantallas móviles. Ofrece una arquitectura innovadora, lo que le permite tener compatibilidad con diferentes protocolos de comunicación. Gracias a sus redes escalables y redundantes garantiza una conectividad ininterrumpida y una máxima seguridad de tus datos.

Como se ha comentado anteriormente, inVendi BMS permite controlar los sistemas que conforman un edificio. Por ello, ofrece muchos beneficios en su gestión. Uno de los beneficios más notables es la optimización energética, gracias al registro de los datos de consumo del edificio. Además, permite controlar varios sistemas diferentes al mismo tiempo para lograr un uso óptimo de los recursos, y tener un sistema de control de las alarmas con datos en tiempo real. Este notifica los errores y, además permite la configuración de recordatorios para el mantenimiento preventivo de equipos y así prolongar su vida útil.
